This repository is a high-fidelity scaffold to recreate the provided dark, glossy, neon-blue portfolio design using React (Vite) + Tailwind CSS.

Getting started
--------------
1. Install dependencies:

   npm install

2. Start dev server:

   npm run dev

Project notes
-------------
- Add project images to `public/assets/projects/` and update `src/data/projects.js`.
- The styles live in `src/styles/globals.css` (Tailwind + custom helpers).
- Follow the PLAN.md for the detailed implementation plan.
